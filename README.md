# Habit Tracker App
> A habit tracker app, where we can track habits.
## Technologies Used
1.  NodeJS
2.  Express
3.  EJS
4.  MongoDB

## Prerequisites
- MongoDB
- Git
- NodeJS
- CLI

## Installation

##### Installing NPM dependencies

`npm install`

##### Then simply start your app

`npm start`

#### The Server should now be running at http://localhost:3000/
